如果您学习过图像预处理教程，您将看到我们正在使用rescale和裁剪函数来准备图像，以及重新格式化图像为CHW、BGR和NCHW。我们还纠正了图像的平均值，通过使用提供的npy文件的计算平均值或静态删除128作为占位符平均值。

您会发现加载预先训练的模型非常简单，只需几行代码即可完成。这里我们展示了这几行:

# 1.读取pb(Protobuf文件)
with open("init_net.pb") as f:
     init_net = f.read()
with open("predict_net.pb") as f:
     predict_net = f.read() 
     
#2. 在工作空间中使用预测函数来加载来自protobufs的blobs。
p = workspace.Predictor(init_net, predict_net)

#3.运行网络得到结果
results = p.run([img])

结果返回的是一个多维的概率数组。本质上，每一行都是一个百分比概率，这个对象与神经网络识别的东西相匹配。当你运行花时，它应该给你超过95%的评价，花是一朵雏菊。

在下面的代码块中，我们正在加载平均文件(如果它存在)和图像，然后预处理图像，将其输入到一个Caffe2卷积神经网络中!

def crop_center(img,cropx,cropy):
    y,x,c = img.shape
    startx = x//2-(cropx//2)
    starty = y//2-(cropy//2)    
    return img[starty:starty+cropy,startx:startx+cropx]

def rescale(img, input_height, input_width):
    print("Original image shape:" + str(img.shape) + " and remember it should be in H, W, C!")
    print("Model's input shape is %dx%d") % (input_height, input_width)
    aspect = img.shape[1]/float(img.shape[0])
    print("Orginal aspect ratio: " + str(aspect))
    if(aspect>1):
        # landscape orientation - wide image
        res = int(aspect * input_height)
        imgScaled = skimage.transform.resize(img, (input_width, res))
    if(aspect<1):
        # portrait orientation - tall image
        res = int(input_width/aspect)
        imgScaled = skimage.transform.resize(img, (res, input_height))
    if(aspect == 1):
        imgScaled = skimage.transform.resize(img, (input_width, input_height))
    pyplot.figure()
    pyplot.imshow(imgScaled)
    pyplot.axis('on')
    pyplot.title('Rescaled image')
    print("New image shape:" + str(imgScaled.shape) + " in HWC")
    return imgScaled

Caffe组件

Blobs and Workspace, Tensors

数据在caffe2中规划为blob。blob只是内存中的一个命名数据块。大多数blobs都包含一个张量(考虑多维数组)，在Python中，它们被转换成numpy数组(numpy是Python的流行数字库，并且已经作为一个先决条件添加了Caffe)

Workspace存储所有的blobs，下面展示如何将blobs填充到workspace中并整合。工作空间在您开始使用它们时初始化它们自己。

from caffe2.python import workspace, model_helper
import numpy as np
# Create random tensor of three dimensions
x = np.random.rand(4, 3, 2)
print(x)
print(x.shape)

workspace.FeedBlob("my_x", x)

x2 = workspace.FetchBlob("my_x")
print(x2)
Nets和Operators

net是一个操作图和。每个操作接受一组输入blob并产生一个或多个输出blob。

下面将建立一个简单的模型，包含以下的组件:

  • 一个FC层
  • 一个Sigmoid激活Softmax
  • 一个交叉熵损失
# Create the input data
data = np.random.rand(16, 100).astype(np.float32)

# Create labels for the data as integers [0, 9].
label = (np.random.rand(16) * 10).astype(np.int32)

workspace.FeedBlob("data", data)
workspace.FeedBlob("label", label)


# Create model using a model helper
m = model_helper.ModelHelper(name="my first net")


#创建两个网络init_net以及exec_net
weight = m.param_init_net.XavierFill([], 'fc_w', shape=[10, 100])
bias = m.param_init_net.ConstantFill([], 'fc_b', shape=[10, ])
fc_1 = m.net.FC(["data", "fc_w", "fc_b"], "fc1")
pred = m.net.Sigmoid(fc_1, "pred")
softmax, loss = m.net.SoftmaxWithLoss([pred, "label"], ["softmax", "loss"])

总结:

  1. 创建输入数据和标签的blob到内存中；注意，数据和标签blobs有第一个维度’ 16 ‘;这是因为模型的输入mini batch的16个样本。 很多Caffe2的操作都可以直接通过ModelHelper一次处理输入的mini batch量

  2. 使用一些操作例如:FC、Sigmoid、SoftmaxWithLost；注意这时候只是创建一个框架，并未运行

  3. Model helper帮助你创建两个网络；m.param_init_net只运行一次，为所有的FC层的参数blobs初始化，训练实际上是在m.net中

net被存储在protobuf结构中,可以调用net.Proto()查看

#查看网络结构
print(m.net.Proto())
#
#查看参数初始化
print(m.param_init_net.Proto())

Caffe2的核心API思想是:使用Python来方便地编写网络来训练您的模型，将这些网络传递给C代码作为序列化的原始缓冲区，然后让C代码运行具有完整性能的网络

运行

首先运行参数初始化,这实际上会将param_init_net的原始缓冲区传递到C++运行时执行;然后训练模型

workspace.RunNetOnce(m.param_init_net)

workspace.CreateNet(m.net)

#创建一次可以多次使用

# Run 100 x 10 iterations
for _ in range(100):
    data = np.random.rand(16, 100).astype(np.float32)
    label = (np.random.rand(16) * 10).astype(np.int32)

    workspace.FeedBlob("data", data)
    workspace.FeedBlob("label", label)

    workspace.RunNet(m.name, 10)   # run for 10 times

请注意，我们只是通过了m.name，而不是net定义本身到RunNet()。由于在工作区中创建了网络，所以我们不需要再次传递这个定义。

运行完成后，我们可以检查存储在输出blobs中的结果

print(workspace.FetchBlob("softmax"))
print(workspace.FetchBlob("loss"))
